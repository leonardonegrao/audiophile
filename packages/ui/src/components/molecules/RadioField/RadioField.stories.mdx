# RadioField

The `RadioField` allows an user to select a value from a list of options.

export const Template = (args) => {
  const [value, setValue] = useState(args.value);
  return (
    <RadioField
      {...args}
      value={value}
      onChange={(newValue) => {
        setValue(newValue);
      }}
    />
  );
};

## Disabled Option

It's possible to disable an option, the disabled option will be grayed out and won't be selectable.
To allow more flexibility, the prop `isOptionDisabled` receives a function that impletements the logic that
determines if an option is disabled or not.

<Canvas>
  <Story

        { id: 3, label: 'Option 3', value: 3 },
      ],
      value: 1,
      isOptionDisabled: (option) => option.id === 2,
