# Button

The main Button component. It has three variants: `primary`, `secondary` and `tertiary`.

The arguments below are only the custom arguments, all the attributes that the `<button>` of HTML5
receives, this component also accepts.

<Story
  name="Default"
  args={{
    children: 'Click me',
    disabled: false,
  }}
>
  {Template.bind({})}
</Story>

export const Template = (args) => <Button {...args} />;
