# RadioField

The `RadioField` is a compound of two components: `RadioField.Radio` and `RadioField.RadioGroup`.

The `RadioField.Radio` is a component for selectable options, it must be used within `RadioField.RadioGroup` to be interactive.

export const Template = (args) => {
  const [value, setValue] = useState(args.defaultValue);
  return (
    <RadioField.RadioGroup defaultValue={value} onChange={(value) => setValue(value)} {...args}>
      {args.options.map((option) => (
        <RadioField.Radio value={option.value}>{option.label}</RadioField.Radio>
      ))}
    </RadioField.RadioGroup>
  );
};

# RadioGroup

The `RadioField.RadioGroup` is the component that allows the `RadioField.Radio` to be dynamic.
All cases require the `RadioField.Radio` to be inside of a `RadioField.RadioGroup`.

This component controls two things: behavior with `defaultValue` and `onChange`, and appearance with `layout` and
`widthBehavior`.
