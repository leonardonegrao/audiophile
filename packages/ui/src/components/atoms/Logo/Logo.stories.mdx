# Logo

The `Logo` component is used to display the logo of the application. It's flexible to be used in different contexts.

<Story name="Default" args={{ color: 'primary' }}>
  {Template.bind({})}
</Story>

The component has some predefined colors, and can receive a custom hexadecimal.
