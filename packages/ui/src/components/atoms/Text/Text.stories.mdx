# Text

The `Text` component must be used to render every textual content. Primarily, should be used with a
specific `variant`, but can be manually set with the options that receives as `props`.

export const Template = (args) => <Text {...args} />;

<Story
  name="Default"
  args={{
    children: 'Text Component',
    variant: 'body',
  }}
>
  {Template.bind({})}
</Story>
